# Hotel Management System

A desktop-based Hotel Management System developed using **Core Java, Swing, JDBC, and MySQL**.  
This project helps manage hotel operations like customer details, rooms, employees, check-in, check-out, and searching room information.

## 📌 Project Overview

The Hotel Management System is a Java Swing application that provides an easy-to-use interface for hotel staff to manage daily hotel activities.  
It connects with a MySQL database using JDBC to store and retrieve data.

## 🚀 Features

- Customer Registration
- Room Management
- Employee Management
- Department Management
- Customer Information Management
- Manager Information
- Check-In and Check-Out System
- Room Search Functionality
- Database Connectivity using JDBC

## 🛠️ Technologies Used

- **Programming Language:** Java
- **GUI:** Java Swing
- **Database:** MySQL
- **Database Connectivity:** JDBC
- **IDE:** IntelliJ IDEA / Eclipse
